## 我们追踪什么

七大油气航运咽喉要道：

| 咽喉要道           | 基线流量（百万桶/日） |
|-------------------|----------------------|
| 霍尔木兹海峡       | 21.0 |
| 马六甲海峡         | 17.2 |
| 苏伊士运河 / SUMED | 7.6  |
| 曼德海峡           | 6.2  |
| 丹麦海峡           | 3.0  |
| 土耳其海峡         | 2.9  |
| 巴拿马运河         | 0.9  |

基线来自 EIA 2023 年《World Oil Transit Chokepoints》出版物。两个咽喉要道（土耳其海峡 + 丹麦海峡）**不**被常见竞品站点覆盖——它们在此处呈现。

## 实时流量如何推导

不是编辑估算。每个发布的流量数字都基于真实观测值校准，6 小时刷新一次：

1. **Portwatch DWT** — IMF / 世界银行的船舶追踪聚合，按船舶类别生成每个咽喉要道的载重吨位。
2. **AIS 中继风险匹配** — 我们的 AIS 接入（Railway 中继）为距咽喉要道 500 公里以内的船舶打标，附带观测到的中断信号（GPS 干扰、AIS 关闭、锚地聚类、红/橙 GDACS 警报）。
3. **流量估算** = 当前 DWT ÷ 365 天基线 DWT，以基线的百分比表示。
4. **置信度带** = 船舶样本规模与中断信号密度的函数。

刷新节奏：`energy:chokepoint-flows:v1`（6h Railway cron）+ `supply_chain:chokepoints:v4` 热点探测（30 分钟）。

## 状态徽章推导

每个咽喉要道都有一个派生的公开徽章：`normal | restricted | disrupted | closed`。该徽章是以下输入的确定性函数：

- 流量比率（当前百万桶/日 ÷ 基线）
- 500 公里内的活跃 AIS 中断计数
- 战争风险等级（低 / 中 / 高 / 极高）
- 活跃航行警告（NGA）

如果没有可见的底层证据字段，我们不会发布一个单纯的结论标签（"制裁封锁"、"伊朗关闭"等）。

## 每个公开数字都附带其依据

每个咽喉要道卡片通过标准的归属页脚暴露其来源溯源：

- 来源类型（`ais` 代表 Portwatch + AIS，`regulator` 代表 EIA 基线）
- 样本规模（N 个 AIS 观测）
- 方法（`AIS-DWT calibrated`）
- 置信度带（`high | medium | low`）
- 数据新鲜度（`Xh ago`）

该页脚是一个 `HTMLElement`，带有 `data-attr-source`、`data-attr-n`、`data-attr-confidence` 属性，以便代理（MCP 客户端、抓取器、分析工具）可以提取与人眼可读相同的来源溯源。

## 已知限制

- AIS 信号在严重干扰的地区（曼德海峡、霍尔木兹海峡）会衰减。缺口以降低的置信度呈现，而非被合成填补。
- 基线是年度平均值。季节性波动（如苏伊士冬季/夏季）尚未建模。
- 在条带层级没有按大宗商品拆分（原油 vs 成品油 vs LNG）。将通过管道登记表交叉链接在后续版本中提供。
