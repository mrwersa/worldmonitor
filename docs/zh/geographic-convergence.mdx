最有价值的情报信号之一，是**多个独立数据流在同一地理区域聚合**。这往往先于重大事件发生。

## 工作原理

系统维护一个实时地理单元网格（1° x 1° 分辨率）。每个单元追踪四类事件：

| 事件类型 | 来源 | 检测方法 |
|------------|--------|-----------------|
| **抗议** | ACLED/GDELT | 直接地理定位 |
| **军用航班** | OpenSky | ADS-B 位置 |
| **海军舰船** | AIS 流 | 船舶位置 |
| **地震** | USGS | 震中位置 |

当**3 种或更多不同事件类型**在 24 小时窗口内出现在同一单元时，会生成一个**聚合警报**。

## 聚合评分

```
type_score = event_types × 25      # Max 100 (4 types)
count_boost = min(25, total_events × 2)
convergence_score = min(100, type_score + count_boost)
```

## 告警阈值

| 聚合类型数 | 评分区间 | 告警等级 |
|-----------------|-------------|-------------|
| **4 类** | 80-100 | 严重 |
| **3 类** | 60-80 | 高 |
| **3 类**（低计数） | 40-60 | 中 |

## 示例场景

**台海集结**

- 单元：`25°N, 121°E`
- 事件：军用航班（3）、海军舰船（2）、抗议（1）
- 评分：75 + 12 = 87（严重）
- 信号："Geographic Convergence (3 types) - military flights, naval vessels, protests"

**中东闪点**

- 单元：`32°N, 35°E`
- 事件：军用航班（5）、抗议（8）、地震（1）
- 评分：75 + 25 = 100（严重）
- 信号：多条活动流在该区域聚合

## 为何重要

单一数据点往往只是噪声。但当**抗议爆发、军事资产重新部署、地震监测仪检测到异常**同时发生在同一地点时，就值得关注 — 无论任何单一来源是否报告了危机。
